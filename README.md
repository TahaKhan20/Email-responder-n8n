# 📧 AI Email Responder – n8n

## Overview

This n8n workflow automatically monitors incoming emails and generates a **personalized, properly formatted reply draft** using GPT.  
The draft is saved for manual review before sending.

---

## Workflow

1. **Email Trigger** (Gmail / IMAP)  
2. Extract sender, subject, and body  
3. Send content to GPT with structured prompt  
4. Generate formatted reply  
5. Create email draft  

---

## Features

- 📥 Automatic email detection  
- 🧠 GPT-based contextual understanding  
- ✍️ Personalized reply generation  
- 📝 Professional formatting (greeting, body, closing)  
- 🔐 Draft-only (human review before sending)  

---

## Tech Stack

- n8n  
- OpenAI GPT / OpenRouter API 
- Gmail

---

## Use Cases

- Client inquiry handling  
- Freelancing responses  
- Business communication
